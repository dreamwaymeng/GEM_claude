# GEM Quark Model

A C++ project for calculating multiquark state masses using the quark potential model within the Gaussian Expansion Method (GEM) framework.

## Features

- Calculate masses of conventional mesons ($q\bar{q}$)
- Calculate masses of baryons ($qqq$)
- Calculate masses of exotic multiquark states (tetraquarks, pentaquarks)
